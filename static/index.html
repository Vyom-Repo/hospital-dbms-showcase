Hospital DBMS Showcase

Advanced PostgreSQL Database Engineering Showcase

A production-inspired Hospital Management System built to demonstrate advanced PostgreSQL concepts including schema design, concurrency control, stored procedures, triggers, indexing strategies, query optimization, and scalable backend architecture.

Live Demo 🐘 PostgreSQL (Neon) ⚡ FastAPI (Render) ▲ Frontend (Vercel) 📊 1.4M+ Records 📜 MIT License

🎯 Purpose — Why This Project Exists

This application is not intended to be a complete Hospital ERP. Its primary objective is to demonstrate advanced Database Management System (DBMS) concepts using PostgreSQL while keeping the frontend intentionally lightweight.

The user interface acts as a transparent window into the database, allowing visitors to explore how SQL, transactions, constraints, views, and backend logic work together seamlessly under high workload scenarios.

🎯 Project Goals

  • 🛡️
    Demonstrate production-grade PostgreSQL engineering with clean 3NF schema design and strict constraints.
  • ⚡
    Showcase database performance optimization on large scale datasets (~1.4M rows) with live EXPLAIN ANALYZE.
  • 🔒
    Illustrate concurrency-safe transaction design using raw SQL, row-level FOR UPDATE locking, and triggers.
